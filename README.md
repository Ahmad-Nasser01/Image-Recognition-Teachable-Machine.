# Image-Recognition-Teachable-Machine.
# Animal Classifier (Lion vs. Tiger)

A simple Python project that uses machine learning to identify whether an image is a lion or a tiger. The model was trained using Google's Teachable Machine.

## What's Inside This Repository
* `keras_model.h5`: The trained model file.
* `labels.txt`: A text file containing the labels (Lion and Tiger).
* `Image_Classification.ipynb`: The Python notebook (Google Colab code).
* `output.png`: A screenshot showing the code working successfully.

## How It Works
1. **Training:** I uploaded pictures of lions and tigers to Teachable Machine and trained the model.
2. **Exporting:** I downloaded the trained model in Keras format (`.h5`).
3. **Coding:** I wrote a simple Python script in Google Colab to load the model, process a test image, and predict the correct animal.
